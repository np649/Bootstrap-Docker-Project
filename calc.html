How the Calulator program demonstrates the key components of Object Orientated Programming

Encapsulation - The calculator program that we created demonstrates encapsulation when you look at the calculation.py class. The function has the “__” in front of the name which denotes that encapsulation is being used

Inheritance - The calculator program also demonstrates that it is using inheritance. Whenever the calculator is performing a calculation, the class for the operator (addition, subtraction, division, etc) inherits the methods and data from the Calculation class.

Polymorphism - The topic of polymorphism can be implemented into most programs you make, you just have to have a single function name that is used for different types.

Abstraction - You can demonstrate the concept of abstraction in your project by using an abstract class that allows you to create a blueprint for future classes that are created.
